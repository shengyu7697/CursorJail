CursorJail
==========

## Introduction
CursorJail is a tool that limit your cursor into a rectangle.  
You can use CursorJail in these situations:
- Multiple monitor: keep cursor inside a single monitor.
- Gaming: keep your cursor inside a game.

## System Requirement
Windows 7, XP

## Develop Environment
Visual studio 2010
